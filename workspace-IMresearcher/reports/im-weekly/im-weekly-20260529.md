## 一、行业动态与发展总结

### 1. 近期政策变动

本周全国层面的政策主线仍是“人工智能+制造”和科技金融协同。5月29日，央行强调用好科技创新和技术改造再贷款等政策工具，指向科创企业、技术改造项目和先进制造场景继续获得政策性金融支持。对智能制造企业而言，这类政策不是泛化利好，而是把融资支持落到研发、中试、设备更新、产线改造和数字化系统投入。

新华网科技本周披露，两部门发文布局人工智能计量能力建设，重点解决AI应用中的“测不准”和“数据荒”问题。智能制造的质量检测、工业视觉、机器人运控、工业大模型和预测性维护都依赖可度量、可复现、可验证的数据基础。计量体系完善后，工业AI项目更容易形成验收标准，也有利于银行围绕合同、验收、回款和设备资产进行授信判断。

人形机器人和具身智能继续进入标准化治理阶段。近期全国首个人形机器人全生命周期管理服务平台发布，为机器人产品赋予唯一编码，覆盖研发生产、市场准入、销售使用和报废回收等环节。该信号意味着机器人产业从样机展示转向可追溯、可监管、可规模交付，银行可更清楚地识别整机、零部件、售后维保和场景运营中的真实交易。

综合影响闭环: 政策信号指向科技金融、AI计量、机器人全生命周期管理和制造业设备更新。经营含义是企业会增加研发、中试、检测认证、产线升级和供应链备货；金融需求集中在流动资金、设备融资、票据贴现、订单融资、保理、现金管理和跨境结算。客户经理应围绕“是否进入政策清单、是否有设备更新项目、是否形成可验收订单、是否有核心客户回款路径”四个问题开展拜访。

### 2. 政府规划与产业方向

国家层面的产业方向已经从单点自动化转向“AI+工业场景”的系统升级。机器人、工业软件、工业互联网、机器视觉、数字孪生、边缘计算和智能物流不再孤立发展，而是共同服务于降本、提质、缩短交付周期和提升安全生产能力。

设备更新仍是制造业投资的重要抓手。对数控机床、工业机器人、检测装备、工业软件和智能仓储企业来说，客户采购决策会更多绑定政策补贴、再贷款、技改项目和绿色制造要求。具备可复制行业方案、可验收项目经验和稳定售后能力的企业，更值得银行提前介入。

资本市场也在强化“硬科技+真实收入”导向。iFinD公告语义检索显示，工业智能机器人在3C、汽车、自动化设备、新能源、半导体、工程机械和生物医药等工业场景渗透提升，AMR、协作机器人和机器视觉正在成为工厂提效工具。银行关注点应从估值故事转向订单、回款、毛利率、客户集中度和产业链议价能力。

### 3. 行业发展趋势

智能制造行业处于结构性上行阶段，但内部明显分化。具身智能、人形机器人、机器视觉、智能质检、工业AI和智能物流仍是高景气赛道；传统自动化装备和项目制系统集成面临价格竞争、验收周期和应收账款压力；工业软件从单点MES/PLM向研产供销一体化、工业数据治理和AI智能体迁移。

本周36氪快讯显示，存储、算力和工业数据基础设施继续受AI需求拉动；工控网本周重点内容集中在工业AI、机器人灵巧手精密检测、工业巡检具身智能、MES与AI融合、预测性维护和边缘AI工控机。行业机会正在从“买一台设备”转向“设备+软件+数据+服务”的组合交付。

银行更应关注三类企业: 第一，有量产能力和核心客户订单的整机或关键部件企业；第二，进入汽车零部件、新能源、半导体、医药美妆、电子制造等高景气客户体系的装备和视觉企业；第三，掌握回款账户、票据流和上下游交易数据的链主或平台型企业。风险端重点关注研发费用前置、应收账款、存货、客户集中度、质保责任和订单真实性。

### 4. 上下游产业链动态

上游关键环节包括伺服电机、减速器、轴承、控制器、传感器、视觉模组、边缘计算硬件、工业连接器和工业软件。上游企业通常需要提前备料、持续研发和客户认证，适合普惠金融科创贷、短贷、银票贴现、商票e贴和知识产权增信等方向。

中游企业包括机器人整机、数控装备、机器视觉、智能仓储、系统集成和工业软件厂商。项目制交付从合同签订、采购备货、现场实施到验收回款存在时间差，适合订单融资、国内信用证开证及融资、境内保函、付融通和资产池。

下游需求主要来自汽车、新能源、电子制造、医药美妆、仓储物流、公共安全巡检、电力巡检和半导体封测。资金占用较大的环节是设备更新项目、批量备货、长账期应收和跨境交付。核心企业一旦形成稳定采购，可带动零部件、材料、检测认证、维保和物流服务商批量获客。

## 二、浙江地区动态与区域机会

### 1. 浙江政策与政府规划

浙江本周最清晰的区域信号来自绍兴人工智能赋能产业升级。人民网浙江5月27日报道，绍兴各地正在依据《绍兴市人工智能产业发展行动方案（2025—2027年）》推进产业升级，重点发展AI应用芯片、工业软件、智能视觉、智能家居、智能机器人和智能制造场景应用。绍兴的纺织印染、黄酒、集成电路、装备制造和新材料基础较强，AI落地更偏向存量工厂改造、视觉质检、能耗优化、排产和设备预测维护。

杭州具身智能仍是浙江智能制造最强热点。5月1日起施行的《杭州市促进具身智能机器人产业发展条例》建立了产业边界、场景开放、数据治理、伦理安全和产业服务框架。杭州已经集聚宇树科技、云深处科技、强脑科技等企业，形成整机、运动控制、脑机接口、灵巧手、场景应用和资本市场联动。

宁波继续承担“AI+制造”的量产和场景验证功能。宁波制造业门类齐全，汽车零部件、装备制造、精密加工、新材料和港口物流场景丰富，适合机器人、机器视觉、工业软件和智能物流企业做行业方案复制。对银行而言，杭甬绍三地分别对应研发链主、量产配套和工厂改造三类客群。

### 2. 浙江重点产业与区域机会

第一，杭州具身智能和机器人产业链。宇树科技、云深处科技代表整机和行业应用方向，强脑科技代表脑机接口和智能仿生方向。杭州的机会不只在头部企业本身，还在零部件、算法训练、数据采集、中试验证、检测认证、场景运营和经销服务企业。

第二，绍兴“AI+制造”改造链。绍兴纺织印染、家居、装备和电子产业基础厚，AI视觉质检、工业软件、节能降耗和柔性排产更容易形成可验收项目。客户经理可围绕柯桥纺织印染、上虞新材料、越城集成电路和诸暨装备制造建立扫客清单。

第三，宁波智能装备和精密制造链。宁波适合关注智能制造装备、轴承检测、汽车零部件自动化、智能物流和工业AI应用。项目制装备企业通常有合同、验收和回款时间差，是订单融资、票据、保函和现金管理的重点客群。

第四，温州、台州、衢州零部件链。伺服、电机、泵阀、轴承、模具、精密加工和工业连接件是机器人和智能装备的底层供应。订单节奏快、客户分散、票据使用多，适合普惠金融场景化方案、银票贴现、商票e贴和付融通批量切入。

### 3. 浙江上下游产业链动态

浙江上游优势在轴承、伺服、电机、传感器、精密加工、工业连接器、功能材料和通用机械。中游优势在杭州具身智能整机、宁波智能制造装备、绍兴工业软件和AI改造、台州温州零部件制造。下游覆盖汽车零部件、新能源、纺织印染、医药美妆、电子制造、仓储物流和公共巡检。

资金占用集中在四类场景: 机器人整机企业的研发和备货，装备集成企业的项目垫资，零部件供应商的原材料采购和账期，制造企业设备更新和软件系统采购。适合批量获客的链条是“杭州机器人链主+宁波/温州/台州零部件供应商+绍兴/宁波工厂应用场景”。

### 4. 银行展业机会

1. 围绕杭州具身智能链主企业，切入数字财资、资产池、付融通、商票e贴、平安薪和跨境支付结算，服务研发扩张、供应链付款和全球化销售。

2. 围绕绍兴AI+制造改造项目，切入科技创新和技术更新改造再贷款、平安租赁、普惠金融科创贷、银票贴现和慧收款，服务视觉质检、节能改造和工业软件采购。

3. 围绕宁波智能装备企业，切入订单融资、国内信用证开证及融资、境内保函、银票极速贴现和资产池，解决项目交付和验收回款周期。

4. 围绕温州、台州、衢州零部件供应商，依托核心客户订单和票据流，推进付融通、商票保贴、商票e贴、普惠金融场景化方案和短贷。

## 三、重点企业推荐

### 企业名称: 杭州宇树科技股份有限公司

- **推荐等级:** 高
- **所属行业:** 具身智能机器人、人形机器人
- **所在地区:** 杭州
- **产业链位置:** 中游整机厂/平台型
- **推荐方向:** 流动资金 / 供应链金融 / 票据 / 现金管理 / 跨境结算 / 代发 / 综合金融

**企业关键信息:**
- **基本情况:** 公司总部位于杭州，主营四足机器人、人形机器人、机械臂等通用机器人产品，处于具身智能整机和运动控制核心环节。公开信息显示，公司IPO进程持续受到市场关注，产品在消费、科研、展示和行业应用场景均有布局。
- **高层与团队背景:** 创始人兼CEO王兴兴公开资料显示为浙江宁波余姚人，本科就读浙江理工大学机械电子工程相关专业，研究生毕业于上海大学机械工程专业，曾有大疆工作经历，硕士阶段即开发四足机器人原型；导师、CFO等信息待核实。
- **银行关注点:** 创始人技术背景强，产品工程化和品牌传播能力突出；上市前后供应链规范、资金归集、跨境收付、代发和核心供应商融资需求会提升。

**推荐理由:**
- 杭州具身智能政策和产业生态持续强化，宇树处在浙江机器人链主位置。
- 人形机器人进入标准化和全生命周期管理阶段，整机企业的量产、质保和供应链管理价值上升。
- 公司若继续推进IPO和全球销售，银行可提前建立主办行与综合金融关系。

**银行展业机会:**
量产和新品迭代带来零部件备货、供应商付款和研发周转需求，可用付融通、商票e贴、银票无感贴现和短贷切入；海外销售和全球渠道需要跨境支付结算、跨境贸易金融、外币存款和平安避险；上市阶段可配置数字财资、资产池、平安薪和资本市场融资服务。

**推荐产品组合:**
建议以数字财资和付融通作为主推，解决资金归集、供应商付款和账期管理；配套商票e贴、银票无感贴现、跨境支付结算、平安薪和资本市场融资服务。额度、期限、费率待沟通。

**客户经理切入话术:**
“我们关注到具身智能进入标准化和规模交付阶段，宇树的供应链付款、海外收款和上市规范化会同步变复杂。我们可以先从核心供应商、回款账户和跨境收付三件事帮贵司做一版资金方案。”

**风险提示:**
具身智能商业化仍在爬坡，需核实订单真实性、产能利用率、客户集中度、海外收入、应收账款、研发投入和知识产权风险。

### 企业名称: 杭州云深处科技股份有限公司

- **推荐等级:** 高
- **所属行业:** 四足机器人、轮足机器人、人形机器人、工业巡检机器人
- **所在地区:** 杭州
- **产业链位置:** 中游整机厂/行业应用服务商
- **推荐方向:** IPO前综合金融 / 订单融资 / 供应链金融 / 票据 / 现金管理 / 代发

**企业关键信息:**
- **基本情况:** 公司成立于2017年，聚焦具身智能机器人行业应用，产品用于电力巡检、消防应急、工业巡检等To B场景。公开信息显示，公司科创板IPO申请已获受理，募投方向涉及算法模型、本体与解决方案研发及产业化基地建设。
- **高层与团队背景:** 公开资料显示，公司由浙江大学控制科学与工程学院相关教师朱秋国及校友李超等创立，李超担任联合创始人兼CTO；具体导师、CFO和财务团队信息未披露。
- **银行关注点:** To B行业应用场景回款路径相对可核验，IPO阶段对账户管理、票据管理、募集资金规范、代发和资本市场服务要求提高。

**推荐理由:**
- 云深处处于杭州具身智能行业应用核心梯队，场景集中在电力、消防和工业巡检，订单可验证性相对较强。
- IPO受理带来资金规范、账户体系、募投项目和供应链付款需求。
- 机器人全生命周期管理平台强化了行业合规和质量追溯，利好具备真实场景和产品交付能力的企业。

**银行展业机会:**
项目制订单从合同到验收回款存在周期，可切入订单融资、国内信用证开证及融资、银票贴现和商票e贴；IPO阶段可配置数字财资、资产池、平安薪和资本市场融资；供应商付款可评估付融通和供应链金融。

**推荐产品组合:**
建议以数字财资和订单融资作为主推，解决上市阶段资金规范和项目交付资金占用；配套资产池、商票贴现、付融通、平安薪和资本市场融资服务。额度、期限、费率待沟通。

**客户经理切入话术:**
“云深处进入IPO关键阶段后，资金归集、项目回款、票据和供应商付款都需要更规范。我们可以先从订单合同、回款账户和现有票据结构入手，梳理一套上市阶段银行服务清单。”

**风险提示:**
需核实IPO进度、订单结构、客户集中度、应收账款账龄、研发费用、质保责任和核心团队稳定性。

### 企业名称: 杭州强脑科技有限公司

- **推荐等级:** 中高
- **所属行业:** 脑机接口、智能仿生、灵巧手和康复机器人
- **所在地区:** 杭州
- **产业链位置:** 上游核心技术/中游智能硬件
- **推荐方向:** 科创融资 / 设备融资 / 现金管理 / 代发 / 综合金融

**企业关键信息:**
- **基本情况:** 公司公开资料显示专注非侵入式脑机接口、智能仿生手、康复设备和人机交互产品，技术与具身智能、灵巧操作和医疗康复场景相关。
- **高层与团队背景:** 创始人韩璧丞公开资料显示曾在哈佛大学相关实验室从事脑机接口研究并创业；具体导师、CFO和核心产业化团队信息待核实。
