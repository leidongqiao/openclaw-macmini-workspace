## 一、行业动态与发展总结

### 1. 近期政策变动

**（1）国家人工智能应用中试基地（具身智能）在杭州正式揭牌**（5月19日）

5月19日，国家人工智能应用中试基地（具身智能）在杭州高新区（滨江）正式揭牌。这是全国唯一面向具身智能领域的国家人工智能应用中试基地。同日，浙江省委副书记、省长刘捷主持召开省政府专题会议，研究部署浙江省国家人工智能应用中试基地建设工作。去年8月国务院印发《关于深入实施"人工智能+"行动的意见》，明确提出布局建设一批国家人工智能应用中试基地，此次杭州具身智能基地落地，标志着具身智能从实验室走向规模化的关键节点。

**（2）杭州发布《人工智能创新高地建设2026年行动方案》**（2月）

杭州明确提出到2026年AI核心产业营收规模超5500亿元，年增速保持20%以上。这是全国首个以"争创第一城"为目标的城市级AI产业规划。

**（3）吉林省助力具身智能等未来产业人才引育**（5月中旬）

吉林省人社厅印发政策清单，从引才育才、职称评审、资金补助等维度推出20项措施，对具身智能相关单位吸纳高校毕业生给予补贴。

**综合影响闭环：**

具身智能和人形机器人从"概念验证"正式进入"规模化产业化"阶段。国家级中试基地的落地意味着技术标准、测试认证、应用示范将统一，企业不再各自摸索。对于银行而言，这意味着三类业务机会值得重点关注：①中试基地配套的基础设施建设、设备采购需要项目贷款；②具身智能企业在规模化扩张阶段的流动资金、科创贷需求；③产业链上下游（减速器、伺服电机、传感器、工业软件）企业的订单融资和供应链金融。

### 2. 政府规划与产业方向

**（1）工信部推动重点装备领域建设数字化车间、智能工厂**

工信部印发《"十四五"信息化和工业化深度融合发展规划》，围绕机械、汽车、航空、航天、船舶、电子、电力等重点装备领域，建设数字化车间和智能工厂，构建面向装备全生命周期的数字孪生系统，推进基于模型的系统工程（MBSE）规模应用。这为制造企业明确了数字化升级的技术路径。

**（2）河北推进能源装备高质量发展（2026-2030年）**

河北省发改委等联合印发方案，明确"十五五"时期能源装备制造业总产值达2000亿元以上，重点在可再生能源、新型电力系统和智慧能源装备领域发力。

**（3）中国已建成加氢站突破550座**

截至2026年4月，国内累计建成加氢站557座，现有在建及规划加氢站达177座。燃料电池汽车产业链的持续扩张对上游装备制造企业带来订单机会。

**（4）华为AI+制造方案**（5月19日）

华为发布"打破烟囱、让工厂化身数智生命体"的AI+制造方案，推动Agent技术与工厂MES/ERP/PLM系统深度融合。这意味着工业软件正在从"信息化"向"智能化"演进，对MES、SCADA、工业互联网平台供应商既是机会也是挑战。

### 3. 行业发展趋势

**（1）机器人行业**

据CMR产业联盟统计，2026年一季度全球累计推出27款AGV/AMR新品，其中国内企业占24款，涵盖无人叉车、料箱机器人、复合机器人、轮式人形机器人等各品类。AGV/AMR物流领域出货量同比增长超30%，市场规模突破百亿美元。人形机器人方面，全国首条万级人形机器人自动产线已落地，产业规模正从"验证期"进入"量产期"。

**（2）数控机床/工业母机**

国产机床正加速摆脱"组装依赖"，朝着"核心自主、智能引领、生态协同"方向迈进。从CIMT展会技术突破来看，五轴联动加工中心、AI+工业母机创新成果成为焦点。国产替代窗口期正在打开，核心关注科德数控等五轴联动整机企业和核心部件企业。

**（3）工业软件**

2026年PLM系统竞争格局正在重塑。Nvidia Omniverse推动工业软件与CAD/CAE/PLM/MES/SCADA系统深度集成，物理AI正在成为工业软件平台化的新引擎。中型制造企业在MES选型中面临信息孤岛、实施周期长、运维滞后三大痛点，国产MES厂商在本地化服务方面具备优势。

**（4）行业判断**

- 智能制造整体处于**上行期**，具身智能/人形机器人是最大结构性增量
- 工业软件国产替代、数字孪生、AI质检是确定性强的细分赛道
- 数控机床国产替代处于加速期，核心部件企业弹性更大
- AGV/AMR从"规模扩张"转向"高质量发展"，头部企业集中度提升

### 4. 上下游产业链动态

**上游：** 传感器、伺服电机、减速器等核心零部件企业订单增长显著；加氢站建设带动燃料电池系统装备需求。

**中游：** 工业机器人整机厂、AGV/AMR企业扩产积极；国产机床厂产能利用率提升；MES/PLM等工业软件厂商加速交付。

**下游：** 汽车制造（新能源）、3C电子、半导体是智能制造装备最大采购方；"5G+工业互联网"在多地落地，工厂数字化改造需求持续释放。

**资金占用加大环节：** 中游装备制造企业应收账款增加（下游客户付款周期拉长）；国产替代初期研发投入大，现金流压力值得关注。

**供应链金融切入点：** 传感器/伺服电机等上游零部件企业可用订单融资切入；中游系统集成商可用商票保贴、付融通缓解应收账款压力；下游汽车/3C制造企业可用数字财资、资产池优化资金管理。

## 二、浙江地区动态与区域机会

### 1. 浙江政策与政府规划

**（1）国家人工智能应用中试基地（具身智能）落地杭州滨江**

这是全国唯一面向具身智能领域的国家级中试基地。5月19日揭牌后，浙江省长刘捷随即召开专题会议部署中试基地建设。杭州具身智能与智能终端制造业集群已集聚700余家企业，2025年集群规上工业产值突破1000亿元。杭州发布行动方案明确2026年AI核心产业营收超5500亿元。

**银行机会：** 中试基地配套的场地建设、设备采购、测试认证服务存在项目融资需求；700余家集聚企业中的扩产型、研发型企业存在科创贷、流动资金贷需求；企业设备采购可用科技创新和技术更新改造再贷款切入。

**（2）浙江"415X"先进制造业集群新增5个千亿级核心区**

2025年新增的5个千亿级核心区包括：高端软件集群核心区滨江区、人工智能集群核心区西湖区、新能源汽车及零部件集群核心区钱塘区、现代家具与智能家电集群核心区慈溪市、高端新材料集群核心区衢州智造新城。

**（3）2026年度浙江省"415X"集群新质生产力项目启动申报**

申报截止时间11月26日，重点覆盖具身智能、工业软件、新能源汽车零部件、高端装备等领域，获批项目将获得财政专项资金支持。

**（4）平湖依托本地化智能制造优势，锚定人形机器人核心零部件及专用机器人领域**

平湖培育了长三角地区极具竞争力的机器人加工配套能力，加工精度可达微米级。

**（5）宁波人工智能产业产值上半年达328亿元，同比增长19.2%**

宁波抢抓人工智能发展机遇，在AI赋能制造方面持续发力。宁波也是工信部首批"5G+工业互联网"融合应用试点城市。

### 3. 浙江上下游产业链动态

**上游（浙江）：** 平湖的精密加工配套能力为人形机器人提供核心零部件；宁波的传感器、温州的低压电器为智能制造装备提供基础元器件；衢州的化工新材料为特种机器人、传感器提供材料支撑。

**中游（浙江）：** 杭州集聚了700余家具身智能与智能终端企业，覆盖人形机器人整机、四足机器人、AGV/AMR、工业软件等全链条；宁波的"5G+工业互联网"平台企业为浙江工厂数字化改造提供平台服务。

**下游（浙江）：** 慈溪智能家电、绍兴纺织、温州电气、台州汽车零部件等传统制造行业正在加速智能化改造，对MES系统、工业机器人、AGV等需求旺盛。

**资金占用加大环节：** 中游机器人/AGV企业扩张期研发和产线投入大，现金流压力大；下游传统制造企业数字化改造前期资金密集。

### 4. 银行展业机会

1. **具身智能产业集群批量获客：** 杭州滨江/西湖区域700余家具身智能企业正处于从实验室走向产业化的关键阶段，中试基地揭牌后将加速企业扩产。可用科创贷、普惠金融科创贷批量切入。
2. **"415X"集群项目申报配套融资：** 2026年度新质生产力项目申报窗口已开，获批企业将获得财政支持，可配套提供项目贷款、设备融资。
3. **传统制造企业数字化改造：** 慈溪智能家电、绍兴纺织、温州电气等行业改造需求旺盛，可用科技创新和技术更新改造再贷款、MES系统供应商订单融资切入。
4. **平湖精密加工配套企业：** 人形机器人核心零部件需求爆发，平湖微米级精密加工企业有望获得更多订单，可用订单融资、商票保贴支持其快速扩产。
5. **宁波"5G+工业互联网"生态：** 首批试点城市政策红利释放，参与5G+工厂改造的平台企业和终端制造企业都存在设备采购、系统部署、流动资金需求。

## 三、重点企业推荐

### 企业名称：浙江宇树科技有限公司

**推荐等级：** 高
**所属行业：** 智能机器人/具身智能
**所在地区：** 浙江杭州（滨江）
**产业链位置：** 中游（整机制造）
**推荐方向：** 科创贷 / 供应链金融 / 现金管理 / 跨境结算

**企业关键信息：**
- **基本情况：** 宇树科技是四足机器人和轮式人形机器人领域的头部企业，产品在国内市场份额稳居高位。2025年10月完成B2轮融资，融资规模9亿元人民币，成为具身智能领域年度最大单笔融资。总部杭州，在全球机器人竞赛和工业巡检场景持续拓展。
- **高层与团队背景：** 创始人兼CEO王兴兴，浙江大学硕士毕业，在校期间即开始四足机器人研发。创业前曾在大疆等科技企业有过技术积累，在机器人运动控制算法方面有深厚积累。技术团队核心成员多来自浙大、清华等高校机器人实验室。
- **银行关注点：** 创始团队技术背景扎实，融资能力强（B2轮9亿元），在具身智能赛道具备先发优势。产品已覆盖消费级和工业级市场，商业化路径清晰。

**推荐理由：**
- 杭州具身智能产业集群核心企业，全国唯一具身智能中试基地直接受益方
- B2轮融资9亿元验证了资本市场认可度，企业进入快速扩张期
- 四足/轮式人形机器人产品处于产业化爆发前夜，订单和产能需求将快速上升
- 作为链主企业可带动上下游精密零部件、传感器等配套企业批量获客

**银行展业机会：**
- 扩产/产线建设需要项目贷款、设备融资
- 研发周期长、投入大，可用普惠金融科创贷补充流动资金
- 随着海外订单增长，存在跨境结算、外币存款、跨境资金管理需求
- 作为链主企业可带动供应链金融（订单融资、付融通）批量覆盖上下游

**推荐产品组合：**
建议以**普惠金融科创贷**切入，解决企业研发扩产期的资金缺口；配套使用**数字财资**优化资金管理、**跨境支付结算**支持出海业务；待产能释放后，引入**银票极速贴现**和**供应链金融**服务上下游。额度和费率待沟通。

**客户经理切入话术：**
"宇树最近拿到9亿融资，产线扩张和研发肯定需要大量资金支持。平安银行在科技创新和技术更新改造再贷款方面有专项政策，加上我们科创贷审批快、额度灵活，非常适合您这个阶段的资金使用需求。另外如果后续有海外客户订单，我们的跨境结算也能帮上忙。"

**风险提示：**
具身智能行业商业化仍处于早期，大规模量产和盈利能力有待验证；竞争加剧（小米、小鹏等企业也在布局），市场份额可能被蚕食。仍需结合征信、流水、财报、合同、客户结构和实地尽调确认。

### 企业名称：浙江晶盛机电股份有限公司

**推荐等级：** 高
**所属行业：** 半导体装备/光伏智能装备
**所在地区：** 浙江绍兴（上虞）
**产业链位置：** 上游（设备制造商）
**推荐方向：** 项目贷款 / 订单融资 / 票据 / 现金管理

**企业关键信息：**
- **基本情况：** 晶盛机电是国内晶体生长设备龙头，业务涵盖半导体硅片设备、光伏长晶设备等。在半导体大硅片国产化进程中占据核心地位，是国内少数能批量供应8-12英寸半导体硅片生长设备的企业。公司总部位于绍兴上虞，2025年营收和利润保持高增长。
- **高层与团队背景：** 董事长邱敏秀、总经理何俊，核心团队多具备中科院、浙江大学等科研院所背景，在晶体生长技术领域深耕多年。公司技术团队与多家半导体材料研究所有深度合作。
- **银行关注点：** 国产半导体设备替代的核心受益方，订单饱满（长晶设备供不应求），下游客户包括中环股份等头部企业。资金回款周期较长（下游半导体企业付款慢），应收账款管理值得关注。

**推荐理由：**
- 半导体设备国产替代是确定性最强的赛道之一，晶盛在长晶设备领域市占率领先
- 下游半导体扩产带来持续订单，营收确定性高
- 下游客户付款周期长，应收账款规模大，金融需求明确

**银行展业机会：**
- 订单饱满但回款慢，可用商票保贴、付融通缓解应收账款压力
- 扩产投入可用项目贷款、设备融资
- 大客户集中度高，可用国内信用证降低交易风险
