## 一、行业动态与发展总结

### 1. 近期政策变动

本周最重要的政策信号是国家发展改革委、国家能源局发布发改能源〔2026〕688号文，推动“多用户绿电直连”。政策将风电、光伏、生物质等新能源发电通过专用线路和变电设施向多个法人用户供给绿电，明确适用工业园区、零碳园区、增量配电网、重点用能企业、出口外向型企业及其上下游企业，并支持分布式光伏通过集中汇流方式参与。项目按“以荷定源”规划，年自发自用电量占总可用发电量比例不低于60%，占总用电量比例不低于30%，2030年前不低于35%。

政策同时明确项目投资主体、储能与运营平台建设、计量结算、绿电溯源等关键机制。项目主责单位原则上由电源方与负荷方合资，也可由电源方、负荷方或园区平台投资；连接线路、变电设施、储能及运营平台原则上由主责单位投资建设。项目需通过配置储能、集控管理、负荷调节和多能互补提升就近消纳能力；绿电溯源结果可按小时级新能源发用电量匹配，并纳入绿证核发交易体系。

综合影响闭环：政策把“绿电资产”从单一电源项目扩展为“园区能源基础设施+储能+结算运营平台”。对制造业园区、算力中心、出口型工厂和新能源设备企业而言，经营含义是用能成本、碳足迹、绿证与客户合规要求可以被一体化管理；金融需求将集中在项目贷款、设备融资、储能系统采购、园区运营平台现金管理、上下游供应链融资和绿电服务结算；平安银行可围绕科技创新和技术更新改造再贷款、银团融资、平安租赁、数字财资、资产池、国内信用证及保理/付融通形成组合方案，客户经理应优先与园区平台、头部用能企业和储能/逆变器/电力设备供应商沟通绿电直连项目储备。

### 2. 政府规划与产业方向

国家层面继续把新型电力系统、绿电消纳、算电协同和制造业设备更新作为新能源主线。688号文明确优先支持算力设施、绿色氢氨醇等新兴产业和未来产业开展绿电直连，说明新能源不再只是发电端投资，而是与算力、氢氨醇、先进制造、出口低碳合规绑定。

制造业投资方向也在变化。2026年以来逾60家A股上市公司披露扩产或大额投资计划，新能源、AI算力产业链、半导体是本轮扩产最集中的方向。锂电产业链一季度披露85个扩产项目，总投资超过2200亿元；头部企业扩产更偏高端储能电池、大容量电芯、材料一体化和本地化产能。银行应关注“真需求驱动的高端扩产”，而不是上一轮低端重复产能。

### 3. 行业发展趋势

新能源行业整体处于“分化复苏+结构升级”阶段。光伏端，央国企组件集采从低价导向转向效率和资产价值导向，2026年以来主要央国企组件集采规模达到78.4GW，已有59GW公布结果，高效组件、BC、HJT等高效标段占比接近半数，说明低效产能仍有压力，但高效组件、逆变器、储能一体化方案具备更强订单韧性。

储能端景气度高于传统动力电池。4月全球储能电芯CR3、CR5分别为45.8%和60.5%，头部厂商产能利用率维持高位，国内独立储能、共享储能集中交付，海外工商业储能和长时储能需求持续释放。大容量电芯、钠离子电池、AIDC数据中心储能成为本轮技术关键词。

风险方面，光伏组件价格仍处低位，组件企业盈利修复不均衡；锂电扩产若集中释放，可能在两年后形成阶段性供需压力；储能出海还面临出口退税下降、海外认证、本地化服务和汇率波动。银行授信应区别对待：优先选择技术门槛、订单确定性、现金流和海外服务能力较强的企业。

### 4. 上下游产业链动态

上游材料端，储能电芯需求回暖带动磷酸铁锂、前驱体、电解液、隔膜等材料企业新一轮投资，但价格和库存仍需跟踪。中游环节，组件采购进入高效化，逆变器和储能系统集成从单一硬件销售转向“光储一体化+运维+交付”方案。下游端，央国企电站、工业园区、算力中心、出口型制造企业对绿电、绿证、储能和电力调度的需求明显增强。

资金占用最大的环节主要在电池材料扩产、储能系统集成备货、组件/逆变器交付和海外项目履约。适合银行切入的产品包括项目贷款、设备融资、平安租赁、国内信用证、银票贴现、保理/付融通、订单融资、跨境结算和外汇避险。

## 二、浙江地区动态与区域机会

### 1. 浙江政策与政府规划

浙江本周的核心机会来自“绿电直连+算电协同+制造业集群”。杭州、宁波、嘉兴入选首批城域“毫秒用算”专项行动名单，宁波作为制造业大市，正在把算力低时延应用与新能源电力底座结合起来。宁波到2025年底电源总装机3466万千瓦、清洁能源装机1720万千瓦，分布式光伏装机928万千瓦，规模居全国第一；当地正在加快建设120万千瓦新型储能设施，并推进甘浙特高压直流工程受端配套、500千伏线路和新型电力系统建设。

这对本地企业的影响很直接：工厂屋顶光伏、园区储能、算力中心用电、绿证溯源和虚拟电厂会从“可选项”变为制造业客户的成本与合规工具。银行机会集中在园区综合授信、储能设备采购融资、光伏/储能供应商票据与保理、算力中心项目融资、跨行账户集中管理和代发薪酬服务。

### 2. 浙江重点产业与区域机会

浙江新能源产业基础适合围绕三类区域扫客：第一，杭州的光伏逆变器、储能电力电子、风电整机和数字能源服务企业；第二，宁波的分布式光伏、海上风电、海缆、动力电池材料和算电协同场景；第三，嘉兴、绍兴、湖州、金华等制造业园区的工商业储能、屋顶光伏和绿电直连需求。

宁波的机会尤其明确。分布式光伏与制造业屋顶资源结合紧密，海上风电、核电、光伏和储能形成“源网荷储数”协同空间；象山南田岛算力中心、象山输变电工程和周边海上风电/核电项目，为绿电算力、储能和海缆设备打开长期需求。

### 3. 浙江上下游产业链动态

浙江上游以电池材料、海缆材料、电力电子元器件和装备制造为主；中游包括逆变器、风电整机、储能系统集成、海缆和新能源运营；下游是工商业屋顶光伏、园区绿电直连、算力中心、港口和新能源车工厂。

资金占用增加的环节包括储能/逆变器订单交付前备货、海缆和风电项目履约、材料企业技改扩产、工商业储能项目投资。可批量获客的抓手是核心企业上下游：以禾迈股份带动逆变器和储能供应链，以运达股份带动风电零部件和运维链，以东方电缆带动海上风电工程配套，以容百科技带动材料供应商和客户账期管理。

### 4. 银行展业机会

1. 园区绿电直连与工商业储能：优先对接杭州、宁波、嘉兴工业园区和零碳园区，提供项目贷款、平安租赁、科技创新和技术更新改造再贷款、数字财资。

2. 光储设备链供应链金融：围绕逆变器、储能系统、组件和电力设备订单，切入国内信用证、银票贴现、商票e贴、订单融资、付融通。

3. 宁波算电协同与海上风电链：关注算力中心配套电网、储能、海缆和风电设备，提供银团融资、项目融资、保函、资产池和现金管理。

4. 储能出海与低碳合规：对出口型储能、逆变器、电池材料企业，配置跨境支付结算、人民币国际证+福费廷、外币存款、跨境资金管理和平安避险。

5. 专精特新企业批量经营：对浙江新能源链上中小企业，用普惠金融科创贷、普惠金融信用贷、普惠金融担保贷和平安薪做组合切入。

## 三、重点企业推荐

### 企业名称：杭州禾迈电力电子股份有限公司

- **推荐等级：** 高
- **所属行业：** 光伏逆变器、储能电力电子
- **所在地区：** 杭州
- **产业链位置：** 中游
- **推荐方向：** 流动资金 / 供应链金融 / 跨境结算 / 票据 / 现金管理 / 代发

**企业关键信息:**
- **基本情况:** 成立于2012年，主营微型逆变器、储能逆变器及电力变换设备，科创板上市公司，处于光伏和储能系统核心电力电子环节。
- **高层与团队背景:** 公开资料显示，核心创始人杨波为博士研究生学历，曾在浙江大学电气工程学院从事博士后及教学科研工作，具备电力电子技术背景；其他高管具体导师和完整履历未披露，待核实。
- **银行关注点:** 技术型创始团队有利于判断产品迭代和海外客户认可度；若订单增长，供应商备货、海外回款、汇率避险和研发投入会形成连续金融需求。

**推荐理由:**
- 多用户绿电直连和分布式光伏集中汇流政策利好逆变器、储能逆变器和能源管理设备。
- 央国企光伏集采转向高效组件和光储融合，逆变器及储能控制能力的重要性提升。
- 杭州入选“毫秒用算”专项行动，未来算力中心及工商业储能需要更高质量电力电子设备。
- 企业具备科创属性和杭州本地产业链带动价值，适合客户经理优先跟进。

**银行展业机会:**
订单交付和海外销售增长会带来备货资金、供应商账期、出口结算、汇率避险和多账户管理需求。平安银行可从供应链票据、订单融资、跨境结算和数字财资切入，同时关注其上下游供应商批量融资。

**推荐产品组合:**
建议以数字财资和跨境支付结算作为主推产品，解决企业多账户和海外收付款管理；配套银票极速贴现、国内信用证、订单融资和平安避险，用于供应商付款、订单备货和汇率风险管理。额度、期限、费率和准入条件待沟通。

**客户经理切入话术:**
“近期绿电直连和工商业储能政策明显利好电力电子企业，我们想了解贵司今年逆变器和储能订单交付节奏，看看能否在供应商付款、出口结算和汇率避险上提前配一套方案。”

**风险提示:**
光伏行业价格竞争仍激烈，海外认证、渠道库存和汇率波动需关注；仍需结合征信、流水、财报、订单、客户结构和实地尽调确认。

### 企业名称：运达能源科技集团股份有限公司

- **推荐等级：** 高
- **所属行业：** 风电整机、智慧能源服务
- **所在地区：** 杭州
- **产业链位置：** 中游 / 服务商
- **推荐方向：** 项目融资 / 供应链金融 / 保函 / 票据 / 现金管理

**企业关键信息:**
- **基本情况:** 浙江省大型国有上市企业，长期从事新能源技术研究、风电整机及新能源服务，是浙江风电产业链核心企业之一。
- **高层与团队背景:** 公开资料显示，公司高管团队包含博士、正高级工程师、省部级科技奖获得者和享受国务院特殊津贴专家等技术管理人才；具体个人导师、完整职业路径待核实。
- **银行关注点:** 国资背景和技术团队有助于参与大型风电、绿电直连、园区综合能源项目；项目制业务对保函、票据、供应链和现金管理需求较强。

**推荐理由:**
- 国家政策支持风电等新能源就近消纳，风电企业可参与园区绿电直连和综合能源服务。
- 浙江沿海和宁波象山等区域海上风电、绿电算力需求提升，带来整机、运维和综合能源服务机会。
- 项目制订单通常带动零部件供应商、施工方和运维服务商，适合批量供应链经营。

**银行展业机会:**
风电项目投标、履约、设备采购和应收账款回款周期较长，适合以保函、国内信用证、银票贴现、商票e贴、付融通和资产池组合切入；对重大项目可联动银团融资和平安租赁。

**推荐产品组合:**
建议以境内保函和国内信用证开证及融资作为主推产品，覆盖投标履约和设备采购；配套银票无感贴现、付融通、资产池和数字财资，用于供应商即时收款、票据资产管理和集团资金统筹。额度、期限、费率和准入条件待沟通。

**客户经理切入话术:**
“绿电直连政策把风电、储能、园区负荷放到一个项目主体里，贵司后续项目会牵动不少供应商。我们可以先从保函、票据和供应商融资做一套链式方案，帮助项目交付更顺。”

**风险提示:**
风电项目周期长，受招投标价格、原材料成本、并网消纳和应收账款影响；需核实项目回款、担保条件和合同真实性。

### 企业名称：宁波容百新能源科技股份有限公司

- **推荐等级：** 高
- **所属行业：** 锂电池正极材料
- **所在地区：** 宁波
- **产业链位置：** 上游 / 中游材料
- **推荐方向：** 流动资金 / 项目融资 / 跨境结算 / 供应链金融 / 现金管理

**企业关键信息:**
- **基本情况:** 科创板上市公司，主营锂电池正极材料及前驱体研发、生产和销售，产品应用于新能源汽车动力电池、储能和电子产品等领域。
- **高层与团队背景:** 公开资料显示，创始人、董事长白厚善毕业于中南大学有色冶金相关专业，并具有东北大学硕士学习经历，长期从事有色金属和电池材料产业；其他高管完整履历待核实。
- **银行关注点:** 材料企业扩产、原材料采购和客户账期占用明显，管理层产业背景有助于判断技术路线和客户资源，但需持续关注行业周期和合规风险。

**推荐理由:**
- 储能锂电需求回暖，头部电芯企业订单排产较满，正极材料和前驱体存在结构性机会。
- 公开报道显示，公司拟投建磷酸铁锂前驱体及相关项目，顺应储能用磷酸铁锂需求。

**银行展业机会:**
材料采购、扩产技改和客户账期会形成大额营运资金需求；海外客户或原材料进口会带来跨境结算、外币存款和避险需求。银行可围绕项目建设、应收账款、票据和跨境资金管理切入。

**推荐产品组合:**
建议以科技创新和技术更新改造再贷款或项目贷款作为主推产品，支持技改扩产；配套付融通、商票保贴、国内信用证、跨境贸易金融和外币存款，用于材料采购、客户账期和跨境收付。额度、期限、费率和准入条件待沟通。

**客户经理切入话术:**
