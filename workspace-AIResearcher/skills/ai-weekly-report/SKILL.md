# AI 行业周报生成 Skill

汇总近 7 天 AI 行业日报精华，生成周报（动态 + 商机），推送至飞书群。

## 何时使用

- 用户说"生成周报"、"出周报"、"AI 周报"、"行业周报"
- 定时任务触发（每周五下午 5 点）
- 补发历史周报

## 核心原则

1. **汇总而非新建**：基于近 7 天日报A+B的内容提炼，不重新抓取原始信息
2. **地域聚焦**：**商机地图、行动清单、企业动态仅限浙江本地**（杭州/宁波/温州/绍兴/嘉兴/湖州/金华/台州/丽水/衢州/舟山）；长三角异动作为补充参考
3. **TOP5-8精选**：近 7 天最具实质影响的5-8条事件
4. **商机汇总**：按产品维度汇总近 7 天可介入的商机
5. **单篇控制在2000字以内**
6. **独立运行**：周报由本 skill 独立负责，日报 skill 中不包含周报内容

## 工作流

### 第一步：读取近 7 天日报

**时间范围：上周六至本周五，共 7 期日报。**

**从「AI 行研」知识库搜索该时间范围内的 AI-daily 文档**：

```
feishu_search_doc_wiki action=search
query=AI-daily
filter={doc_types: ['DOCX'], create_time: {start: '7天前', end: '今天'}}
```

读取这些日报文档内容：
```
feishu_fetch_doc doc_id=<文档token>
```

从结果中提取所有事件，按日期和地域分组。

### 第二步：筛选 TOP5-8 要闻

从近 7 天事件中筛选出 TOP5-8 条最具实质影响的要闻：

**筛选标准（按权重排序）：**
1. 国家级政策/百亿级投资（★★★★★）
2. 省级政策/十亿级融资/重大订单（★★★★）
3. 产能变动/价格变动/重大产品发布（★★★）

**排除：**
- 已标注 status=closed 且无后续影响的事件
- 重要性 ≤2 的一般动态
- 纯股市行情（无产业事件支撑）

### 第三步：浙江地市动态

按浙江地市汇总近 7 天关键变化：

```

[嘉兴] ...
[台州] ...
[温州] ...
[湖州] ...
[金华] ...
[丽水] ...
[衢州] ...
[舟山] ...
```

**标注关键节点：**
- 产线投产/设备搬入
- 政策细则落地
- 企业融资/上市
- 订单/定点公告

### 第四步：浙江企业动态

从事件日志中提取所有 `[浙江]` 标签的事件，按企业整理：

```

```

### 第八步：写入知识库

**重要：每次生成都覆盖当前同名文件（AI-weekly-YYYYMMDD），不要有重复日期的文档。**

**步骤：**

1. **列出知识库根目录节点，查找是否已有同名文档**：
   ```bash
   lark-cli wiki nodes list --params '{"space_id":"7630717889183534041","page_size":50}'
   ```
   从返回结果中搜索 title 为 `AI-weekly-YYYYMMDD` 且 `parent_node_token` 为 `""`（根目录）的节点，提取 `obj_token`。

2. **如果找到同名文档（根目录下）**：
   - 提取 obj_token
   - 使用 `lark-cli docs +update --doc <obj_token> --as bot --mode overwrite --markdown '<内容>'` 覆盖内容
   - 输出文档链接：`https://www.feishu.cn/wiki/<node_token>`

3. **如果未找到同名文档**：
   - 使用以下命令在**根目录**以**机器人身份**创建：
     ```bash
     lark-cli wiki +node-create --as bot \
       --space-id "7630717889183534041" \
       --title "AI-weekly-YYYYMMDD" \
       --obj-type "docx"
     ```
   - 从返回结果提取 `obj_token`（用于内容更新）和 `node_token`（用于 URL）
   - 使用 `lark-cli docs +update --doc <obj_token> --as bot --mode overwrite --markdown '<内容>'` 写入内容
   - 输出文档链接：`https://www.feishu.cn/wiki/<node_token>`

lark-cli wiki nodes list --params '{"space_id":"7630717889183534041"}' | \
