# 互联网行业商机周报

覆盖周期：2026.05.15-05.22

资料来源：国家数据局、国家发改委发布会、市场监管总局发布会、人民日报/人民网、新华社/新华网、科技日报、证券时报、21世纪经济报道、界面新闻、36氪、财联社、iFinD公告语义检索、SearXNG站内搜索等公开资料。iFinD热点事件接口返回403，已以公告语义检索结果补充。

## 一、行业动态与发展总结

### 1. 近期政策变动

国家数据局印发《2026年数字经济发展工作要点》，将数据要素市场化配置、全国一体化算力网、数据赋能人工智能、数字经济核心企业培育、制造业数字化转型、数据安全治理和数字经济国际合作列为年度重点。这意味着互联网企业的竞争重点正在从流量和用户增长，转向数据资产、算力调度、行业AI应用和合规运营能力。

国家发改委在5月22日发布会上强调推动人工智能与经济社会各行业广泛深度融合，并指导国产大模型加大力度适配国产算力芯片。对AI应用企业而言，国产算力适配会提高自主可控要求，也会推动云服务、MaaS平台、模型推理和行业大模型项目进入更密集的采购周期。

市场监管总局明确将集中整治直播电商广告、引证广告、AI生成广告等违法行为，并推进统一大市场和信用监管工具运用。互联网广告、内容电商、直播电商和AIGC营销公司需要同步加强素材审核、数据留痕、广告合规与内容风控系统建设。

综合影响闭环：政策信号指向“算力网+数据要素+AI应用+合规治理”四条主线。行业经营含义是AI应用商业化仍处上行期，但合规、算力成本和数据安全成为硬约束。金融需求将集中在云资源预付、算力设备采购、政企项目账期、广告合规系统改造、跨境数据与结算合规等场景。平安银行可从短贷、普惠金融科创贷、科技创新和技术更新改造再贷款、资产池、数字财资、付融通、商票e贴、跨境结算和平安避险切入。

### 2. 政府规划与产业方向

国家层面继续把数字经济作为稳增长和新质生产力的重要抓手。一体化算力网、数据基础设施、数据产权登记、公共数据授权运营价格形成机制，将推动数据、网络、算力、能源协同布局。对互联网企业而言，未来的核心资源不只是用户流量，而是可合规调用的数据、稳定低成本的推理算力和可复用的行业场景。

产业方向上，AI不再只停留在大模型发布和参数竞赛，而是进入智能体、MaaS、AI办公、AI营销、AI客服、AI电商、AI编程、AI视频、工业智能体和数据治理平台的商业化阶段。企业级AI服务将从“买软件席位”转向“按任务、按调用量、按Token、按价值结果”付费。

### 3. AI时代互联网行业发展趋势

本周行业处于“AI基础设施景气、SaaS估值承压、应用商业化分化”的阶段。21世纪经济报道跟踪显示，全球SaaS与企业应用软件板块估值明显回落，AI Agent正在冲击传统按席位订阅制。国内软件企业仍具备数据安全、私有化部署、行业定制等壁垒，但必须尽快把AI嵌入业务流程，而不是只做功能插件。

互联网场景中，搜索、广告、电商、办公、客服、内容生产、游戏、跨境获客和企业服务正在被AI重构。更值得银行关注的企业类型包括：拥有稳定企业客户和数据资产的SaaS公司、能把AI变成付费功能的垂直应用公司、服务跨境卖家的平台和支付结算公司、具备算力或MaaS入口的平台型企业，以及能带动开发者和ISV生态的云平台。

主要风险包括：AI生成广告和直播电商监管加强、数据安全与算法备案压力、模型推理成本高于商业化收入、SaaS续费率下滑、政企项目回款周期拉长、海外平台规则和汇率波动。

### 4. AI Token消费生态链与上下游动态

上游方面，证券时报报道三大运营商开始推出Token算力服务，上海电信、联通、移动相继试水按Token计费，算力租赁商也从“卖裸算力”转向“卖Token/模型服务”。这表明推理算力正在成为可计价、可结算、可规模化采购的生产资料。

中游方面，iFinD公告语义检索显示阿里巴巴在海外监管公告中强调AI智能体将由模型生成Token驱动，AI相关收入占云智能集团外部商业化收入较高比例，千问模型、PAI平台、灵骏智算、MaaS能力构成AI+云全栈服务。该信号对浙江尤其重要，因为杭州是阿里云、千问生态和AI开发者生态的重要承载地。

下游方面，企业级AI应用从“提效工具”走向“运营基础设施”。SaaS、私域运营、数字营销、游戏、电商和跨境获客工具都在探索按量计费、任务计费和AI功能付费。现金流压力最大的环节通常是下游AI应用公司：收入端仍在验证付费率，成本端却需要持续支付模型API、云服务、数据标注和研发人员费用。

适合银行切入的环节包括：算力和服务器采购对应平安租赁、科技创新和技术更新改造再贷款；模型API和云资源预付对应短贷、资产池、数字财资；政企和大客户项目账期对应付融通、保理、商票保贴、商票e贴；跨境AI工具和SaaS出海对应跨境支付结算、跨境资金管理、外币存款、人民币国际证+福费廷和平安避险。

## 二、浙江地区动态与区域机会

### 1. 浙江政策与政府规划

杭州在5月19日“开局十五五”新闻发布会上明确提出打造人工智能创新发展第一城，并把平台经济向智能经济跃升作为核心方向。公开信息显示，杭州人工智能核心产业营收已达4600亿元，已集聚阿里千问、DeepSeek等开源基础大模型，以及宇树科技、云深处等具身智能企业，并落地医疗、具身智能等国家级人工智能中试基地。

杭州“十五五”期间将重点推进开源基础大模型、具身智能全产业链、人工智能产业促进条例、国家数据要素综合试验区、国家数据产业集聚区、Token创新基地和结算枢纽建设。杭州还提出培育“296X”先进制造业集群，打造人工智能、视觉智能两个万亿级集群，推动AI工厂、行业大模型和工业智能体平台落地。

这对本地互联网企业的影响很直接：AI企业会获得更多场景、政策、园区和资本关注；平台型企业和SaaS企业会被推动向智能体、数据治理、行业大模型服务升级；跨境电商和数字贸易企业会产生更强的数字结算、合规、汇率管理和海外收款需求。

### 2. 浙江重点产业与区域机会

杭州是本周最值得扫客的核心区域，重点在滨江、余杭、萧山、钱塘等AI和数字经济集聚区。企业类型包括大模型与MaaS平台、企业服务SaaS、AI营销与客服、跨境数字贸易平台、游戏与内容平台、云服务生态商和工业智能体服务商。

宁波、金华义乌、温州等区域适合围绕跨境电商、产业带出海、B2B平台和数字贸易服务做批量获客。浙江制造业数字化基础较强，“人工智能+制造”会带动工业软件、数据治理、工业视觉、AI质检、供应链协同平台的融资和结算需求。

### 3. 浙江AI Token消费生态链与上下游动态

浙江本地链条呈现“杭州模型/云/平台—全省制造和电商场景—跨境数字贸易”的结构。上游是阿里云、算力调度、IDC、数据服务和软硬件服务商；中游是千问、DeepSeek、魔搭社区、企业服务SaaS、数据治理和行业模型平台；下游是电商、直播、跨境电商、游戏、数字营销、工业软件和制造业AI应用。

Token调用和云资源占用最明显的企业包括：大模型与AI平台企业、AI营销/客服/办公SaaS、游戏内容生产与运营企业、跨境卖家获客工具、工业智能体平台。银行可围绕“云资源预付、模型API成本、研发人员扩张、政企项目回款、平台商户结算、跨境收款”六类场景设计方案。

### 4. 银行展业机会

1. 围绕杭州AI开源生态和Token结算枢纽，优先跟进阿里云、DeepSeek及其开发者/ISV生态，用数字财资、资产池、短贷、科创贷和平台数字贷切入。

2. 围绕AI Agent冲击SaaS行业，跟进群核科技、涂鸦智能等企业服务和AI应用公司，关注模型API、研发投入、客户回款和订阅转型压力，主推普惠金融科创贷、短贷、付融通、慧收款。

3. 围绕杭州游戏、内容和AI营销生态，跟进网易杭州等头部平台及上下游CP、广告服务商、云服务商，用数字财资、跨境结算、平安薪、商票e贴和供应链金融扩生态。

4. 围绕义乌、宁波、杭州跨境数字贸易和B2B支付平台，跟进XTransfer、跨境服务商和产业带企业，主推跨境支付结算、跨境资金管理、外币存款、人民币国际证+福费廷和平安避险。
